# Mohammad Zaid - 3D Portfolio

This repository contains the source code for my personal 3D portfolio, built with React, TypeScript, Three.js, React Three Fiber, and GSAP. It features interactive 3D elements, smooth scroll-driven animations, and showcases my career as a GenAI Architect.

## Features

- Responsive, modern one-page portfolio layout.
- 3D character scene rendering powered by React Three Fiber and Three.js.
- GSAP-powered animations and custom cursor interactions.
- Highlighting projects like QuantDSS, FileForge, RepairDeskz, and Job-Pilot.
- Custom aesthetic emphasizing AI and backend expertise.

## Tech Stack

- **Core**: React 18, TypeScript, Vite
- **Animation & 3D**: GSAP, Three.js, `@react-three/fiber`, `@react-three/drei`
- **Other**: `react-icons`, `react-fast-marquee`

## Getting Started

1. Clone the repository:
   ```bash
   git clone git@github.com:mohd98zaid/3D-Portfolio.git
   cd 3D-Portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```
